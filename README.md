## 🤖 Cómo Funciona

El sistema utiliza **Inteligencia Artificial** para proporcionar diagnósticos confiables y guías personalizadas:

### 1️⃣ **Descripción del Problema**
Describes los síntomas o problemas que presenta tu moto:
- Ruidos extraños
- Problemas de arranque
- Consumo excesivo de combustible
- Vibraciones anormales
- etc.

### 2️⃣ **Información del Vehículo**
Proporciona detalles de tu moto:
- Marca y modelo
- Año de fabricación
- Cilindrada
- Tipo de motor (4T, 2T)

### 3️⃣ **Diagnóstico Automático**
La IA analiza la información y proporciona:
- Posibles causas del problema
- Nivel de dificultad de la reparación
- Herramientas necesarias
- Tiempo estimado

### 4️⃣ **Guía de Reparación**
Recibes instrucciones detalladas:
- **Preparación**: Herramientas y precauciones
- **Inspección Visual**: Qué y dónde revisar
- **Pruebas Prácticas**: Procedimientos y mediciones
- **Diagnóstico**: Interpretación de resultados
- **Reparación**: Pasos para solucionar el problema

### 5️⃣ **Implementación**
Ejecuta las soluciones recomendadas con confianza

## 📖 Guía de Uso

### Pasos Básicos

1. **Inicia sesión o crea una cuenta**
2. **Describe los síntomas** de tu moto con el mayor detalle posible
3. **Proporciona información del vehículo**:
   - Marca (Honda, Yamaha, Bajaj, etc.)
   - Modelo específico
   - Año de fabricación
   - Otros datos relevantes

4. **Recibe el diagnóstico** de la IA
5. **Sigue la guía paso a paso** para revisar tu moto
6. **Implementa las soluciones** recomendadas
7. **Guarda tu reporte** para referencia futura

### Consejos para Mejores Resultados

- ✅ Sé lo más específico posible en tu descripción
- ✅ Incluye cuándo comenzó el problema
- ✅ Menciona si ha habido cambios recientes
- ✅ Ten a mano la documentación de tu moto
- ✅ Toma fotos de componentes relevantes
- ✅ Prepara las herramientas antes de comenzar

## 💡 Ejemplos de Uso

### Ejemplo 1: Problema de Arranque
```
Síntoma: "Mi moto no arranca cuando la lluvia"
Información: Honda CB125R, 2022
Resultado: Diagnóstico de humedad en bujías y cables
Solución: Guía para secar y revisar sistema de ignición
```

### Ejemplo 2: Consumo de Combustible Alto
```
Síntoma: "Estoy gastando mucho más gasolina"
Información: Yamaha YZF-R15, 2020
Resultado: Análisis de filtro de aire sucio y carburador desajustado
Solución: Pasos para limpiar y ajustar correctamente
```

### Ejemplo 3: Vibraciones Anormales
```
Síntoma: "La moto vibra cuando aceleró"
Información: Bajaj Pulsar, 2019
Resultado: Identificación de cadena destensada
Solución: Guía para ajustar tensión de cadena
```
